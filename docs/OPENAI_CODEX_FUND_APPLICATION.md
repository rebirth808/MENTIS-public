# OpenAI Codex Open Source Fund Application Notes

This document is a public, non-sensitive summary that can be used to explain MENTIS in open-source program applications.

## Project name

MENTIS — Modular Executive Neural Tooling Intelligence System

## Brief description

MENTIS is a local-first AI orchestration and developer terminal toolkit.

The project aims to provide a unified command-line/TUI workspace where builders can route tasks across different AI models, coordinate specialist agents, manage project context, run coding and research workflows, enforce safety policies, and keep human control over tool execution.

The long-term goal is to make advanced AI-assisted development more transparent, modular, and accessible. Instead of relying on a single opaque assistant, MENTIS provides a structured layer for model routing, agent roles, project memory, command UX, validation, and safe automation.

## Repository status

This public repository is currently documentation-only.

The full working implementation is private while it is being cleaned, audited, and prepared for staged open-source release.
