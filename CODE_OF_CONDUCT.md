# Code of Conduct

MENTIS aims to be a serious, respectful, builder-focused open-source project.

## Expected behavior

- Be respectful.
- Be direct but constructive.
- Discuss ideas with evidence.
- Keep feedback useful.
- Respect safety boundaries.
- Do not pressure maintainers into unsafe releases.

## Unacceptable behavior

- Harassment
- Hate speech
- Doxxing
- Credential theft
- Attempts to introduce malware
- Unsafe automation requests
- Abuse of maintainers or contributors

## Scope

This code of conduct applies to repository discussions, issues, pull requests, and project-related community spaces.
