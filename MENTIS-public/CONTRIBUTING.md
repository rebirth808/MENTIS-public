# Contributing

MENTIS is not yet ready for broad code contributions.

This repository currently accepts:

- feedback on the roadmap
- documentation suggestions
- architecture feedback
- use-case ideas
- safety concerns
- issue reports about unclear public docs

## Future contribution areas

Planned future contribution areas:

- terminal UX
- model routing
- provider integrations
- safety policies
- documentation
- examples
- test fixtures
- developer workflows

## Contribution principles

Contributions should be:

- practical
- safe
- transparent
- well-scoped
- documented
- testable

Please avoid submitting:

- secrets
- private logs
- generated junk
- unrelated automation
- unsafe cyber tooling
- unreviewable large drops
